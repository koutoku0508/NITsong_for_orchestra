## 名工大学歌オーケストラアレンジ
2025年度名工大卒業式及び2026年度名工大入学式の為に編曲した名工大学歌「東海の邦のほまれに」のオーケストラアレンジ．
[一般社団法人名古屋工業会　名古屋工業大学全学同窓会](https://www.nagoya-kogyokai.jp/gakka)を基に編曲されています．
### リポジトリの説明
* 参考打ち込み音源(東海の邦のほまれに.wav)
* ```【さわらない】東海の邦のほまれに.mscz```は触れないこと．
* scoreディレクトリにFull scoreおよび各パートのscore(pdf)を用意しています．
